Panduan Troubleshooting

Temukan solusi untuk masalah umum seperti gagal instalasi, crash aplikasi, atau masalah kompatibilitas perangkat.

Pastikan Anda telah mengaktifkan opsi ‘Sumber Tidak Dikenal’ di pengaturan perangkat. Jika tetap gagal, hapus versi lama aplikasi sebelum mencoba instalasi ulang.

Pastikan perangkat memiliki ruang penyimpanan dan RAM yang cukup. Bersihkan cache aplikasi secara berkala atau lakukan restart perangkat Anda.

Beberapa fitur mungkin tidak berjalan optimal pada perangkat dengan sistem operasi di bawah Android 8.0. Pastikan perangkat Anda mendukung versi terbaru aplikasi.

Pastikan Anda memasukkan Token atau Username dengan benar. Jika masih mengalami kendala, hubungi tim dukungan resmi kami untuk bantuan lebih lanjut.

Jika Anda lupa Token atau Username, kami tidak dapat memulihkan data tersebut karena bersifat pribadi. Untuk menghindari hal ini, selalu salin dan simpan Token Anda di tempat yang aman.

Periksa koneksi internet Anda atau coba muat ulang halaman. Jika tetap tidak bisa diakses, pastikan URL yang Anda gunakan sudah benar atau gunakan browser lain.

Jika file hasil unduhan tidak terbaca, coba unduh ulang file tersebut. Pastikan proses pengunduhan selesai sepenuhnya sebelum membuka file.
